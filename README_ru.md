# Динамика и управление обратным маятником с маховиком

Образовательная и экспериментальная платформа для управления обратным маятником с маховиком на базе микроконтроллерхных модулей [VBCore32G4](https://docs.vbcores.ru/docs/Hardware/vbcore). Проект содержит описание динамики, алгоритмов управления и руководство по работе с аппаратной платформой. 

Маятник способен работать в следующих режимах:
- раскачивание (Energy-Shaping Control)
- стабилизация в вертикальном положениии (LQR)
- торможение в нижнем вертикальном положении (Energy-Shaping Control)

Репозиторий обеспечивает отлично подойдет как для курсов или отдельных проектов по теории управления и робототехнике, так и для тех, кто хочет самостоятельно разобраться в управлении неустойчивыми объектами.

## Ключевые компоненты
- полностью готовая к работе прошивка микроконтроллера
- python-файл для моделирования и синтеза управления
- gui-приложение для работы с маятником

## Структура репозитория

 - **Firmware**
   - `wheel_pendulum_practicum.ino` - прошивка микроконтроллера для управления маятником
   - `rwip_for_gui` - прошивка для работы с GUI-приложением
 - **Python**
   - `model.ipynb` - мат. модели маятника + синтез управлени
 - **rwip_gui** - Windows-совместимое GUI-приложение для работы с маятником. Позволяет отправлять команды, решать LQR-задачи и визуализировать на графиках состояние.

## Установка и использование
Существует два способа работы с маятником: для демонстрации и для лабораторных работ.

### Демонстрация
- Установите и настройте всё необходимое ПО [по инструкции](https://edu.vbcores.ru//docs/rwip-control/preparing/windows#по-для-работы-с-аппаратной-частью). 
- Установите **RWIP_GUI** с помоощью файла [rwip_gui_1.0.0_install.exe](./rwip_gui). Это стандартный мастер установки. Запустите его и следуйте инструкциям на экране.
- Прошейте микроконтроллер прошивкой [rwip_for_gui](./Firmware/rwip_for_gui/)
- Откройте **RWIP_GUI** и используйте его для управления маятником.
